About the Project

The primary objective for Project 1is to implement algorithms to process and align image plates from RGB channels to produce colorful photos. These images were originally taken by Prokudin-Gorskii on red, green, and blue glass plates, and I aligned these three-color channels to obtain the final colored image. A pyramid algorithm is implemented to help process image with higher resolutions, which reduced the computation time and space.

In addition, 4 extra features, including Automatic cropping, Automatic contrasting, Automatic white balance, color remapping, and more precise alignment features, are also implemented to improve image quality.
